# 希沃自动登录 🐾

希沃白板的多账号自动登录工具，支持手动模式（账号弹框 + 点击填入）和自动模式（全自动登录流程）。

## 快速开始

```bash
pip install -r requirements.txt
python seewo_login.py
```

或直接双击 `启动.bat`。

## 预览

| 主界面 | 手动浮窗 | 自动覆层 |
|-------|---------|---------|
| 科目按钮 + 全局模式切换 | 点击手机号/密码 → 点希沃输入框自动填入 | 进度条 + 步骤 + 取消 |

## 功能

- **科目登录** — 点击科目 → 自适应/自动/手动，未保存时弹窗选择（15s 倒计时）
- **手动模式** — 置顶悬浮窗展示打码后的手机号和密码，点击后鼠标点哪就填哪（剪贴板粘贴，零停顿）
- **自动模式** — 全流程自动化：结束进程 → 启动希沃 → 输入账号密码 → 勾选协议 → 登录 → 进入备课
- **账号管理** — 支持多科目账号编辑/保存/三次确认删除，密码默认隐藏
- **坐标方案** — 硬坐标抽取到 JSON 方案文件，支持新增/重命名/删除/鼠标读取坐标
- **Debug 模式** — 不执行实际操作，单纯走流程
- **4K 高 DPI** — 自动适配 HiDPI 显示

## 使用流程

```
启动 → 选择学科 → 弹窗选模式（仅第一次）
├── 自动 → 底部覆层显示进度 → 完成 → 弹窗询问退出
└── 手动 → 悬浮窗展示账号密码
           ├── 点击手机号 → 去点希沃输入框 → 自动填入
           └── 右上角 [自动] → 切换到自动模式
```

## 数据文件

启动后自动生成：

| 文件 | 说明 |
|------|------|
| `seewo_accounts.json` | 账号密码（已 gitignore） |
| `seewo_coordinate_schemes.json` | 坐标方案（首次自动创建默认 4k 方案） |
| `seewo_mode_settings.json` | 模式记忆（每个科目的偏好） |

坐标方案 `4k` 默认值：

| 操作 | X | Y |
|------|---|---|
| 输入账号 | 1900 | 790 |
| 输入密码 | 1900 | 960 |
| 勾选用户协议 | 1850 | 1545 |
| 确认登录 | 1900 | 1120 |
| 进入个人版 | 80 | 2055 |

> 双屏/不同分辨率请自行在「坐标方案」tab 中使用"读取"按钮校准。
